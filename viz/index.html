Search

/
Focus the search box.
Plain text
Substring match across subreddits, topics, subtopics, points of view (inc. keywords + descriptions), and phrases.
/regex/ or re:pat
Regex mode. Input turns violet. Invalid patterns show an error.
sub: cl: r/ pos:
Field scope — restrict a term to one kind, e.g. sub:rent, cl:/housing|afford/, r/boston, pos:shortage (points of view).
text:
Search actual post bodies — e.g. text:landlord, text:/(rent|mortgage).*spike/. Hits show snippets with topic context; click jumps to the nearest area.
-term
Exclude entries that match the term, e.g. rent -afford. Combines with any operator.
Multi-token AND
Space-separated tokens must all match, e.g. cl:housing -tax.
Focus empty input
Shows your last 8 searches; Enter reruns.
Shift+Enter (regex)
Paint every matching topic/sub on the globe. Chip shows count; × or Esc clears.
Click r/* suggestion
Filter the globe to that subreddit (intersects with any drill state).
↑ ↓ Enter
Browse suggestions and jump.

Filters & state

Filter chips
Active subreddit (green) + timeline range (amber) + regex paint (violet) chips appear in the nav header. Each has a × to clear.
⟲ Reset x
Clears drill focus, sub filter, timeline range, regex/text paint, search input, and URL hash — one click (or x) back to fresh.
↗ Share
Copies the current page URL (navigation state is not encoded in the address bar). Use Copy as markdown on the focus card for a link that includes drill and filters.
Browser back / forward
Steps through your drill path (topic ▸ sub ▸ stance) one level at a time. Filter changes replace state in place, so scrubbing the timeline or toggling a sub filter doesn't flood history.
📋 Copy as markdown
Small button on focus + point-of-view cards copies a pasteable summary (title, trend, voice mix, top points of view / evidence samples, deep-link with current drill + filters + query). For notes or writeups.
